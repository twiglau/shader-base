# 矩阵
* 是一种描述 `线性变换` 的 数学工具;
- `线性变换`: (xa, ya)v1空间内 => v2空间内(xb,yb); v1->v2空间的线性变换;
```
xb = A * xa + B * yb + C;
yb = M * xa + N * yb + D;
```  
- 以上变换可以用数学工具来表示: `矩阵`;
```
// 增加一个维度,因为用常数 C, 作为平移;
               A   M   0
(xa, ya, 1) *  B   N   0   =  (xb, yb, 1)
               C   D   1

=> 相当于: 
xb = A * xa + B * ya + C;
yb = M * xa + N * ya + D;
1 =  0 * xa + 0 * ya + 1;

=> 
```
